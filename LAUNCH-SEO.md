# Launch checklist: Search Console & off-page SEO

Complete these steps after deploying the Aura rebrand.

## Google Search Console

1. Verify property: `https://lakeviewvillagetownhome.ca`
2. Submit sitemap: `https://lakeviewvillagetownhome.ca/sitemap.xml`
3. Request indexing for:
   - `/`
   - `/aura-lakeview-village`
   - `/aura-lakeview-village/floor-plans`
   - `/faq`
4. Use **Removals** only if old URLs show wrong snippets after 301s propagate (usually not needed)

## Monitor redirects

Confirm 301s work:

- `/pier-house` → `/aura-lakeview-village`
- `/compare` → `/aura-lakeview-village-vs-resale`
- `/aura-towns` → `/aura-lakeview-village`

## Target keywords (90-day tracking)

Track weekly in GSC + a rank tracker:

- Aura Lakeview Village
- Aura Lakeview Village floor plans
- Aura Lakeview Village price
- Caivan Lakeview Village townhomes
- Lakeview Village townhomes Mississauga

## AI citation monitoring (monthly)

Prompt ChatGPT / Perplexity:

- "What is Aura Lakeview Village price and occupancy?"
- "Aura Lakeview Village floor plans Caivan"

Note which URLs are cited; expand content on gaps.
